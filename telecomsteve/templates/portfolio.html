Cloud Native Portfolio

A selection of my cloud native and infrastructure as code (IaC) programming.

Homelab Research Testbed

Homelab is my personal sandbox for hosting and testing microservices in the cloud. The environment runs an autoscaling instance of Kubernetes. Both the Kubernetes cluster itself and microservices within have been deployed using a mix of Terraform, Helm, Ansible and Bash scripting. Full documentation is located within the repositories wiki pages on github.

telecomsteve.com

Telecomsteve is a site designed to be lightweight, easy to maintain, and run within AWS Elastic Beanstalk using the Python Flask framework. A "News" section within the site employs Python multithreading and caching to also pull in the current top 50 stories from the Hacker News API. With no external fonts or javascript libraries, the site receives a score of 100/100 from Google Page Speed Index on both desktop and mobile.

Gitub Source

Blockchain for OpenWRT

A compact and resource-light blockchain designed to run on the OpenWRT WiFi router platform. Written in the Python, the blockchain uses Python Flask to deliver its API interface which includes functions for managing network peers, issuing blocks, and running network consesus.

Gitub Source

SIP Trunking at Twitter

During periods of early growth, Twitter deployed a number of isolated and non-standard phone systems that had become expensive and difficult to support. To manage the complexity, Twitter hired me to unify its networks and migrate them to cloud deployment. The result was a fully virtualized phone network, combined with a Twilio Cloud SIP trunking. This new design reduced recurring carrier costs 72%.
